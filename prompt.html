トークスクリプトガイドアプリ用 Windsurf プロンプト雛形

プロジェクトの目的

営業担当者がスムーズに電話営業できるよう、トークスクリプトを段階的にナビゲートするアプリを作ります。

トークフロー構造（記法ルール）

  • 各ステップには番号をつける（例：ステップ1）
  • メッセージと選択肢を記述
  • 選択肢に応じて遷移先ステップを明記

記述例

ステップ1: 挨拶と確認

メッセージ: 「○○様でお間違いないでしょうか？」 - はい → ステップ2へ - いいえ → ステップ98へ

ステップ2: 用件伝達

メッセージ: 「○○についてご案内です。お時間よろしいでしょうか？」 - はい → ステップ3へ - いいえ → ステップ98へ

FAQの表示条件

  • ステップ番号 に FAQ表示 フラグを付ける
  • フラグがある場合は、事前に定義されたFAQを表示

注意点

  • HTML/CSS/JSで動作するアプリ想定
  • GitHub Pagesで公開する前提
